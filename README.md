# Angular2Examples

Реализация тестовых заданий в рамках личног изучения Angular 2 в связке с ASP.Net Core

## Необходимо разработать директиву — инпут для ввода MAC-адреса

В инпуте при вводе с клавиатуры: 
* не позволять вводить недопустимые символы, 
* буквы из нижнего регистра переводить в верхний, 
* расставлять двоеточия в нужных местах, 
* русские буквы заменять на аналогичные латинские (А, В, С, Е).
